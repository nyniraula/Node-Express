Creating and Starting Express server:

firstly import the express module.

`import express from 'express'`


then we create a variable that calls the express() func to initialize (creates) the server. Most commonly named app.

`const app = express()`


Finally, we make the app (server) listen to requests (aka start the server)

app.listen(PORT:int)
app.listen(3000)

we dont direclty give it a port number, but in professional practice, we define a PORT const that is injected by the env, and set a edge case for a default port when not found.

We dont wanna give away open ports, that is a security vulnerability.

`import express from 'express'

const PORT = process.env.PORT || 3000

const app = express()

app.listen(PORT)
`


Additionally, we can also pass a function that does something on startup to the listen method.
Maybe send a log of server starting to somewhere.

app.listen(PORT, ()=>{
    console.log(`${new Date().toISOString()}: Server Started, Running on PORT ${PORT}`);
})

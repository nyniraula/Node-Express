# 09. DELETE Requests

Self-explanatory — HTTP method for deleting records.

```js
app.delete("/user/:id", async (request, response) => {
  const parsedId = parseInt(request.params.id);

  if (isNaN(parsedId)) return response.sendStatus(400);

  const users = await readDB();

  const userIdx = users.findIndex((user) => user.id === parsedId);

  if (userIdx === -1) return response.sendStatus(404);

  users.splice(userIdx, 1); // remove that record

  await writeDB(users);

  return response.send(users);
});
```

> `splice(idx, 1)` removes exactly one element at that index. The updated array is what gets sent back and written to the DB.

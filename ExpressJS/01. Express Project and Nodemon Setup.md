# 01. Express Project & Nodemon Setup

## Steps

1. Create a project folder
2. `pnpm init -y` — initializes `package.json`
3. `pnpm add express`
4. `pnpm add -D nodemon` — watches for code changes and auto-restarts the server
   > `-D` = devDependency. It's only needed during development, not in production. Gets listed under `devDependencies` in `package.json` instead of `dependencies`.

## Folder Structure

```
root/
└── src/
    └── index.js
```

## `package.json` Scripts

```json
{
  "scripts": {
    "dev": "nodemon ./src/index.js",
    "start": "nodemon ./src/index.js"
  }
}
```

> `npm run dev` just runs whatever script is named `dev` in `package.json`. Same idea as Vite's built-in dev script — we're just defining it manually here.

## Module Type

Set this in `package.json` to use ES modules (instead of CommonJS):

```json
{
  "type": "module"
}
```

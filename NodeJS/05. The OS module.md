# OS module

```javascript
import os from 'node:os'

os.platform()   // 'linux', 'darwin', 'win32'
os.arch()       // 'x64', 'arm64'
os.cpus()       // array of CPU core objects — length gives you core count
os.totalmem()   // total RAM in bytes
os.freemem()    // available RAM in bytes
os.uptime()     // seconds the OS has been running (not your process — the machine)
os.homedir()    // '/home/val' or 'C:\Users\val'
os.tmpdir()     // '/tmp' on Linux — temp directory path
```

## Use cases
```javascript
os.cpus().length // to see how many cores are available
```

```javascript
os.tmpdir() // Where to temporarily store uploaded files before processing them
```

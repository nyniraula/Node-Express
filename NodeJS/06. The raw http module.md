# The raw http module

```javascript
import http from 'node:http'

const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' })
  res.end('Hello')
})

server.listen(3000, () => console.log('running on 3000'))
```

- **HTTP header** is basically the meta data (status code, origin, data type) explicitly.
- **HTTP body** is the `res.end()`.

`createServer` takes a callback — called the request handler. Every single HTTP request that hits your server runs through this one function. `req` is the incoming request, `res` is how you respond.

It accepts a callback with two params, generally named `req` and `res`. `req` is the request that comes from other devices and `res` is response stream that goes out. The function auto passes them.

## What `req` gives you (to view the request details):
```javascript
req.method   // 'GET', 'POST', 'PUT', 'DELETE'
req.url      // '/users/1?include=posts' — full path + query string
req.headers  // { 'content-type': 'application/json', 'authorization': '...' }
```
